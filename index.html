    UI Tools是什么¶

    UI Tools是一套借助GUI界面帮助开发人员快速创建HTML页面的程序，包含程序版和网页版两个版本。

    软件功能¶

    • 可以进行拖拽改变位置和改变大小的布局块
    • 拖拽组件到布局块
    • 通过设置界面设置组件属性，并预览变化
    • 自动生成通用JavaScript代码
    • 网页版文件树列表保存历史页面
    • 网页版支持导入和导出页面功能
    • 程序版支持左侧文件树展示本地文件列表，监听本地文件变化，实现修改双向同步
